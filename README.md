# 🛒 RetailVision AI

### Customer Behavior Analytics & Forecasting System using Computer Vision

---

## 🚀 Overview

RetailVision AI is an end-to-end **Computer Vision-powered analytics system** designed to monitor, analyze, and predict customer behavior in retail environments.

The system leverages real-time object detection and tracking to generate actionable business insights, helping retailers optimize store layout, staffing, and customer engagement strategies.

---

## 🎯 Key Features

* 🔍 **Real-time Person Detection & Tracking** using YOLOv8
* 🧭 **Zone-Based Customer Movement Analysis**
* 📊 **Customer Behavior Data Pipeline (Video → Structured Data)**
* 📈 **Interactive Analytics Dashboard (Streamlit)**
* 🔮 **Footfall Forecasting using Time-Series Models (ARIMA)**
* 💡 **AI-Driven Business Insights & Recommendations**

---

## 🧠 Tech Stack

* **Programming:** Python
* **Computer Vision:** OpenCV, YOLOv8
* **Data Processing:** Pandas, NumPy
* **Visualization:** Matplotlib, Streamlit
* **Forecasting:** Statsmodels (ARIMA)
* **Deployment:** Streamlit Cloud, GitHub

---

## 📊 System Architecture

```text
Video Input → YOLO Detection → Object Tracking → Data Collection (CSV)
→ Data Processing → Analytics → Forecasting → Streamlit Dashboard
```
